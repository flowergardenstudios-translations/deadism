# [DEADISM](https://modrinth.com/project/deadism)
- DeadiSM is a Minecraft Survival Overhaul datapack which adds Custom Items, Weather and Weapons for players to fight the Zombie Apocalypse!
- This Github Respository is Aimed at allowing the community to create translations for the DeadiSM project. Anyone can contribute to the translation process. We believe this allows us to provide a much more accurate translated experience of our projects to people who are fluent in one language over others.
- To start translating, create a fork of this repository, then download/git pull the contents of this repository, make changes, commit and then create a pull request!
